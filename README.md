# Python Fundamentos — ComandoFix

Repositorio de ejercicios y proyectos del aprendizaje de Python desde cero.
Parte del roadmap hacia **Backend Developer with AI Integration**.

## 🗺️ Roadmap completo

### ✅ Fase 1 — Python sólido
- [x] Variables y tipos de datos
- [x] input() y f-strings
- [x] Operadores y condicionales
- [x] Bucles while y for
- [x] Funciones
- [x] Listas
- [x] Diccionarios
- [x] Proyecto final: Gestor de Contactos por consola

### 🔄 Fase 2 — Programación Orientada a Objetos
- [x] Clases y objetos
- [x] Herencia
- [x] Encapsulamiento
- [x] Manejo de archivos y errores
- [x] Librerías estándar: os, json, datetime
- [x] Entornos virtuales y pip
- [ ] Proyecto final: Gestor de Contactos refactorizado con OOP

### ⏳ Fase 3 — Backend con Python
- [ ] APIs REST y HTTP
- [ ] FastAPI
- [ ] SQL + PostgreSQL
- [ ] Autenticación básica con JWT
- [ ] Variables de entorno y seguridad
- [ ] Proyecto final: API REST funcional desplegada

### ⏳ Fase 4 — Integración de IA
- [ ] Cómo funcionan los LLMs
- [ ] OpenAI API y Anthropic API
- [ ] Prompt engineering
- [ ] LangChain básico
- [ ] Conectar IA a una API REST
- [ ] Proyecto final: backend con IA integrada

### ⏳ Fase 5 — Java
- [ ] Java moderno desde cero
- [ ] OOP en Java
- [ ] Spring Boot
- [ ] Consumir APIs externas
- [ ] Proyecto final: servicio en Java conectado a API Python

### ⏳ Fase 6 — Empleabilidad
- [ ] Portfolio en GitHub con README en inglés
- [ ] LinkedIn optimizado
- [ ] CV técnico de una página
