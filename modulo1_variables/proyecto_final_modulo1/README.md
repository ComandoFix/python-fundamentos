# Gestor de Contactos
Proyecto final del Módulo 1 de Python.

## ¿Qué hace?
Herramienta de gestión de contactos por consola que permite:
- Permite agregar, ver, buscar y eliminar contactos almacenados en memoria durante la sesión.

## Funcionalidades
- Menú de Contactos.
- Agregar contacto.
- Ver contactos.
- Buscar contacto.
- Eliminar contacto.

## Tecnologías
- Python 3.
- Sin librerías externas.

## ¿Qué aprendí?
- Uso combinado de listas y diccionarios.
- Funciones.
- Bucles y condicionales.
- Interacción con el usuario por consola.
